== Description ==

Создаёт XML-feed для загрузки на Яндекс Объявления. Для работы требуется WooCommerce!

== Installation ==

В этом разделе описывается, как установить плагин и заставить его работать.

1. Загрузите папку `xml-for-o-yandex` в `/wp-content/plugins/`
1. Активируйте плагин на странице ‘Плагины’ WordPress
1. Перейдите на страницу Экспорт на Яндекс Объявления чтобы настроить плагин

== Frequently Asked Questions ==

= Какие плагины интернет-магазинов поддерживает Ваш плагин? =

Только WooCommerce.

= Как создать XML фид? =

Подробная инструкция со скриншотами тут
Перейдите в меню Экспорт на Яндекс Объявления. В поле «Автоматическое создание файла» выберите любое значение (кроме «Отключено»). Обязательно нажмите «Сохранить» внизу страницы.
Через 1 — 7 минут (зависит от числа товаров), фид будет сгенерирован и вместо данного сообщения появится ссылка.
